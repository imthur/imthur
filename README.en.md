## About me

I'm **Arthur Aguiar**, a **24-year-old Software Developer** based in **Recife, Brazil**.

I focus on **back-end development**, mainly with **Python, Django, REST APIs and data modeling**, while also building fullstack applications with **React, JavaScript, TypeScript and front-end/back-end integrations**.

My background in **Data Analysis** strongly influences the way I design software: I like well-structured systems, clear business rules, consistent data handling and solutions that solve real problems.

I'm currently pursuing a degree in **Software Analysis and Development at Uniasselvi** and continuously improving my skills in **back-end engineering, application architecture, APIs, automation and fullstack development**.

## Current focus

- Back-end development with **Python, Django and Django REST Framework**
- Building **REST APIs**, authentication flows, permissions and business rules
- Fullstack development with **React, Vite, JavaScript and TypeScript**
- Process automation with **Python**
- Data modeling, organization and processing
- Deployment, CI, observability and complete project structure
