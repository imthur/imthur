## projetos ativos

### unifica gestão

saas de rh e departamento de pessoal para pmes brasileiras. substitui
escritório contábil e ferramentas separadas com uma plataforma única:
folha clt completa (inss progressivo, adicionais legais, encargos
patronais), 13º, férias, rescisão com 6 tipos, portal do colaborador
com self-service, gestão de documentos com alertas de vencimento,
multi-tenant com isolamento por schema e trilha de auditoria completa.

profundidade de cálculo acima da média do mercado: tabela progressiva
real, aviso proporcional (lei 12.506/11), acordo 484-a, súmula 449/tst,
fgts digital e encargos discriminados na folha.

```
status  em produção · em processo de captação de clientes
stack   python · django · drf · postgresql · celery · redis · docker · render
```

### portal multi-tenant de assistentes virtuais com ia

plataforma de agentes conversacionais construída para apoiar suporte
automatizado a sistemas em geral. pipeline de rag com busca híbrida
por embeddings, reescrita contextual da pergunta, cache semântico e
streaming de respostas em tempo real. widget javascript embeddável
com tema personalizável, painel administrativo multi-agente com jwt
e google oauth, e ciclo de qualidade próprio com avaliação automática.

```
status  desenvolvimento ativo · chatbot em produção integrado ao unifica
stack   python · flask · react · node · google gemini api ·
        postgresql · redis · docker · nginx · gunicorn · gcp/oracle vm
```
